## 为什么是 ppt-master

ppt-master 是一套专注于“快、稳、美”的 AI 演示文稿工作台。围绕“主题输入 → 大纲生成 → 模板匹配 → 生成预览”的完整链路，
将复杂流程压缩为顺滑的三步操作，让内容创作回归表达本身。

### 亮点

- **主流大厂模型生态**：模型能力对接主流大厂生态，支持行业化定制与私有化落地。
- **智能大纲**：根据主题、文本或文件智能生成逻辑清晰的大纲结构，可视化编辑随时调整。
- **智能模板推荐**：自动推荐风格匹配的演示模板，快速锁定统一视觉风格。
- **实时生成预览**：流式生成过程实时反馈，缩略图与主画面同步更新。
- **高质量导出**：生成 PPTX 文件，适配演讲、汇报、教学、路演等场景。

## 核心能力

- 主题/文本/文件 → 自动生成结构化大纲
- 大纲可视化编辑，章节/页面/标题级别调整
- 模板推荐与一键选择
- 实时生成预览，生成完成可直接下载 PPTX

## 技术栈

- **框架**：Vue 3 + Vite + TypeScript
- **渲染**：SVG + Canvas 双引擎渲染
- **流式通信**：SSE 实时流式输出
- **压缩与解析**：PPTX 结构解析 + gzip 解压
- **安全防护**：阿里云验证码 (Captcha)
- **边缘计算**：阿里云 ESA 边缘函数 + EdgeKV

## 阿里云能力集成

### 🛡️ 阿里云验证码 (AI Captcha)

项目集成了阿里云智能验证码服务，用于防止恶意请求和自动化攻击：用户点击"生成大纲"按钮时弹出验证码，有效防止机器人刷量、恶意调用 AI 接口，保护 API 配额

### ⚡ 阿里云 ESA 边缘函数构建 (Edge Function)
项目使用阿里云 ESA 边缘函数构建并部署，并使用Nodejs函数实现[短链接服务](https://github.com/hobk/esa-kv-short)，用于 PPT 文件的云端存储与分享：

- **应用场景**：生成 PPT 后，将文件下载链接转换为短链接存储
- **边缘优势**：全球边缘节点就近响应，低延迟高可用

### 💾 阿里云 EdgeKV 数据库

边缘函数内部使用 EdgeKV 作为键值存储，实现短链接映射，数据存储在边缘节点，全球同步



## 工作流

1. 选择输入方式（主题 / 文本 / 文件）
2. 生成大纲并可视化编辑
3. 选择模板并生成 PPTX
