## 为什么是 ppt-master

ppt-master 是一套专注于“快、稳、美”的 AI 演示文稿工作台。围绕“主题输入 → 大纲生成 → 模板匹配 → 生成预览”的完整链路，
将复杂流程压缩为顺滑的三步操作，让内容创作回归表达本身。

### 亮点

- **主流大厂模型生态**：模型能力对接主流大厂生态，支持行业化定制与私有化落地。
- **智能大纲**：根据主题、文本或文件智能生成逻辑清晰的大纲结构，可视化编辑随时调整。
- **智能模板推荐**：自动推荐风格匹配的演示模板，快速锁定统一视觉风格。
- **实时生成预览**：流式生成过程实时反馈，缩略图与主画面同步更新。
- **高质量导出**：生成 PPTX 文件，适配演讲、汇报、教学、路演等场景。

## 核心能力

- 主题/文本/文件 → 自动生成结构化大纲
- 大纲可视化编辑，章节/页面/标题级别调整
- 模板推荐与一键选择
- 实时生成预览，生成完成可直接下载 PPTX

## 技术栈

- **框架**：Vue 3 + Vite + TypeScript
- **渲染**：SVG + Canvas 双引擎渲染
- **流式通信**：SSE 实时流式输出
- **压缩与解析**：PPTX 结构解析 + gzip 解压

## 工作流

1. 选择输入方式（主题 / 文本 / 文件）
2. 生成大纲并可视化编辑
3. 选择模板并生成 PPTX

## 快速开始

```bash
npm install
npm run dev
```
